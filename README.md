StreamX Media Player

A lightweight, browser-based media player built with HTML, CSS and JavaScript.

## What it does

StreamX is a general-purpose client for user-supplied streams. It supports:

- Xtream-compatible server login using your own server URL, username, and password
- Local M3U playlist file upload
- Direct M3U playlist URLs supplied by the user

## Features

- Live TV, movie, and series browsing
- Built-in HLS playback
- Channel and category navigation
- Search across loaded content
- Optional local credential storage in the browser
